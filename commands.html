    ü§´Taboo

    Commands related to the taboo plugin.

    /taboo info - Get rules and how-to-play information about the game

    /taboo start - Starts a new game of taboo. Only one game of taboo is allowed at a time per server

    /taboo stop - Force stop a game of taboo


    üÉè Cards against humanity

    Commands related to the C.A.H plugin.

    /cah info - Get rules and how-to-play information about the game

    /cah start [explicit=true|false default=false] - Starts a new game of C.A.H. One game can run at a time for each channel. Explicit flag enables cards to contain NSFW content

    /cah stop - Force stop a game of C.A.H
